Work and Projects

Down below you will find some of the relevant work and projects I have done.

Java ANN Face Recognition

Made for the course "Fundamentals of Artificial Intelligence". A simple Java solution that was built on a perceptron based on classification system that guesses the emotional state of faces presented as input of 20x20 pixel images. It had four basic emotions: Happy, Sad, Mischievous and Mad. Read more about it here.

IDag och Inatt 2018

The annual career day and banquet for Interaction Technology and Design students at Umeå University. I was a member of the IDag-project group for the 2018 event. Read more about it here.

Java Chat Client

Made for the course "Data Communication and the Internet". This terminal based chat client running on Java is capable of connecting to different name-servers, allowing communication with other users.  Read more about it here.

Using multiple infrared sensors to reduce the risk of occupational injury

Made for the course "Interactivity In Smart Environments". The aim of the project was to deliver a help system to one of the world’s leading manufactures of trucks. The system was built of multiple infrared sensors working together in a smart environment cooperated with external agents to improve the working conditions and reduce the risk of occupational injury for factory workers. Read more about it here.

Harnas Wildlife Foundation 2013

I was a volunteer on Harnas Wildlife Foundation in Namibia at 2013. As a volunteer I was helping to protect and rescued animals that was injured or expelled from encroachement environments and the rehabilitation and care of injured animals. Read more about it here.
